# Library Management System

A console-based Library Management System developed using C++.

## Technologies Used
- C++
- Object Oriented Programming (OOP)
- Data Structures
- Algorithms

## Concepts Applied
- Encapsulation
- Inheritance
- Polymorphism
- Vector Data Structure
- Searching Algorithms
- Sorting Algorithms

## Features
- Add books
- Display books
- Search books
- Delete books
- Borrow and return books
- Sort books by title

## Project Structure

- Book Class: Handles book information.
- Person and User Classes: Demonstrate inheritance.
- Library Class: Manages books operations.
- Main: Controls the program flow.-
